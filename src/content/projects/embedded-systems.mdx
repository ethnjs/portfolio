## the project series

Five hardware projects completed through OPS / IEEE at UCI, progressing from analog circuit
fundamentals through wireless IoT on ESP32 microcontrollers.

| Project | Concepts |
|---|---|
| 555 Piano | Schematic reading, PCB soldering, analog circuit design |
| Ultrasonic Trashcan | ESP32, servo control, HC-SR04 sensor |
| iPODuino | UART protocol, serial interfacing, pull-up resistors |
| Weather Station | I2C (AHT20 + LCD), Wi-Fi library, data pipeline |
| Digital Stopwatch | Hardware interrupts, timer libraries, 7-segment display |
